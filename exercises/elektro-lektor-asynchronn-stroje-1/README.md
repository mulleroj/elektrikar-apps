# Asynchronní Stroje - Standalone verze

Tato aplikace je nyní plně nezávislá a funguje jako jediný soubor `index.html`.

### Jak ji používat:
1. Otevřete soubor `index.html` v libovolném webovém prohlížeči.
2. Není potřeba žádná instalace, žádné `npm install` ani internetové připojení (pokud je Tailwind CSS v mezipaměti nebo nahrazen lokálním souborem).

### Funkce:
- Interaktivní simulace točivého pole a rotoru.
- Výpočty skluzu a otáček v reálném čase.
- Kompletní test s vysvětlením odpovědí.
- Responzivní design pro mobily i dataprojektory.
